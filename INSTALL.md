# Installation Instructions for Blender Quad Remesher

## Prerequisites
- Ensure you have Blender installed. (Version 2.8 or higher)

## Steps to Install
1. **Download the Add-on**: Go to the [Releases page](https://github.com/microdevweb/blender-quad-remesher/releases) and download the latest release.

2. **Open Blender**: Launch Blender on your computer.

3. **Access Preferences**: Go to `Edit > Preferences`.

4. **Install Add-on**: In the Preferences window, navigate to the `Add-ons` tab, then click on the `Installâ€¦` button.
   - Select the downloaded .zip file.

5. **Enable Add-on**: After the installation is complete, find `Quad Remesher` in the list and enable it by checking the checkbox next to it.

6. **Save Preferences**: If you want to keep this add-on enabled for future Blender sessions, click the `Save Preferences` button.

## Usage
- You can access Quad Remesher in the 3D Viewport by pressing `N` to open the side panel and finding the Quad Remesher options.
